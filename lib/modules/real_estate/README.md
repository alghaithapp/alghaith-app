# real_estate

Property listings: buy, sell, rent hubs and listing forms.

## Owns

- `screens/real_estate_deal_hub_screen.dart` — top-level deal type hub
- `screens/real_estate_type_hub_screen.dart` — property type selection
- `screens/real_estate_listings_screen.dart` — browse listings
- `screens/real_estate_sale_intent_screen.dart` — sell intent flow
- `screens/real_estate_form_screen.dart` — create/edit listing
